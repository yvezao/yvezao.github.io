# yvezao.github.io

This repository powers my personal academic homepage at **[https://yvezao.github.io](https://yvezao.github.io)**.

Built with the [Academic Pages](https://github.com/academicpages/academicpages.github.io) Jekyll template (a fork of the [Minimal Mistakes](https://github.com/mmistakes/minimal-mistakes) theme).

## Editing the site

- Personal info & links → `_config.yml`
- Home / About page → `_pages/about.md`
- CV page → `_pages/cv.md`
- Publications → `_publications/*.md`
- Profile photo → `images/profile.jpg`
- Resume PDF → `files/CV.pdf`

After editing, commit and push to `main`. GitHub Pages rebuilds in 1–2 minutes.

See `docs/MAINTENANCE.md` (in Chinese) for a more detailed maintenance & deployment guide.
